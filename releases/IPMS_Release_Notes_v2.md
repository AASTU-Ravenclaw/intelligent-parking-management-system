# Release Notes - v2 (Final Release)
**Date:** December 2025
**Baseline Reference:** BL2

### Changes from v1.0
This version includes implementations of approved Change Requests (CR-01, CR-02, CR-03).

- **Fee Calculation (CR-02):** Added logic to display estimated fees based on vehicle type.
- **Logout Functionality (CR-01):** Added a functional Logout button to the dashboard.
- **UI Enhancement (CR-03):** Updated the dashboard theme color to Light Blue for better visibility.
- **Documentation:** Finalized all SCM documents (Audit Report, Change Logs).
